# Safety And Consent

逍遥族TTS 面向本地、授权、个人或团队内部的声音克隆工作流。

## Intended Use

- 克隆自己的声音
- 克隆已明确授权的声音
- 为短视频、课程、动画、播客、内部素材生成配音
- 给 agents 或自动化工具提供本地 TTS 能力

## Do Not Use For

- 未经授权克隆他人声音
- 冒充真实人物进行欺骗、诈骗或误导
- 生成违法、骚扰、仇恨、诽谤或侵犯隐私的内容
- 绕过平台或法律对声音合成内容的披露要求

## Product Notes

当前 CLI 会保存 `consent` 字段，默认是 `self`。后续如果做 UI 或商业版本，可以把授权确认、用途记录、生成历史和水印/披露策略做得更完整。
