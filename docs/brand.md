# 逍遥族TTS Brand

## Positioning

逍遥族TTS 是一个本地优先的声音克隆工具：用户录入自己的声音，形成可复用的声音档案，然后在 Web UI 或 CLI/Agent 工作流中快速生成配音。

品牌气质：自由、可信、本地、创作者工具、Agent-ready。

## Logo Concept

第一版 Logo 使用「逍」作为记忆锚点，结合两条声音波形和一个小光点，表达：

- 声音档案可复用
- 本地隐私和掌控感
- 一键生成配音
- 适合创作者和自动化工具调用

文件：

- `assets/brand/xiaoyaozu-tts-logo.svg`
- `assets/brand/xiaoyaozu-tts-mark.svg`

## Suggested Product Surfaces

## Voice

中文表达建议偏直接、可信，不要过度娱乐化。

示例：

- 新建声音档案
- 选择声音
- 输入目标文案
- 生成配音
- 本地保存，不上传云端
- Agent 可调用
